# Sales Report Mini App


## Prerequisite
1. php 8.2^

## Installation
1. Clone the repo
2. Run `composer install`
3. Create `.env` and set database
4. Run migrations and seed:
   php artisan migrate --seed

## Run prject

1. Run `npm run dev` to complie ccs
2. Run `php artisan serve`
